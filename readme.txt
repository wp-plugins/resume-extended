Create and maintain your resume from within wordpress.

== Description ==

This version only creates a page that contains your resume.  It's buggy and not intended for production use.

= Planned Features =

* Edit and Maintain previously created resumes
* Privacy controls
* Resume sections:
	* Publications
	* Projects
	* References
* Export to:
	* PDF
	* Microsoft Word .doc and .docx
	* Text
	* enhanced XHTML
	* XML Resume Library

* Import from:
	* PDF
	* Microsoft Word .doc and .docx
	* Text
	* hResume
		* this includes LinkedIn
	* XML Resume Library


== Installation ==

1. Upload `resume-extended` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Create a new resume under 'Resume' section
1. Get a job and make installing this plugin moot.
